Input in Nim

In Nim, you can read input from the standard input (stdin) using the readLine procedure. This function reads a line of text from the user and returns it as a string.

# Read a line from standard input
let name = readLine(stdin)

# Convert to integer if needed
let age = parseInt(readLine(stdin))

Key points about input in Nim:

  • readLine(stdin) returns a string
  • To read numbers, you need to convert the string using functions like parseInt, parseFloat, etc.
  • The stdin parameter specifies the input stream (usually the keyboard)
  • Always consider error handling for invalid input in real applications

Output in Nim

Nim uses the echo statement for output. It's similar to print in Python or println in other languages. The echo statement automatically adds a newline at the end.

# Basic output
echo "Hello, World!"

# Output multiple values
echo "Name: ", name, " Age: ", age

# Format strings (Nim 2.0+)
echo "User: {name}, Age: {age}"

Key features of echo:

  • Automatically converts values to strings
  • Adds a newline character at the end
  • Can output multiple values separated by commas
  • For output without a newline, use stdout.write

Numeric Operators

Nim supports all standard arithmetic operators for numeric operations. These operators work with integers, floats, and other numeric types.

# Basic arithmetic
let a = 10
let b = 3

echo "Addition: ", a + b # 13
echo "Subtraction: ", a - b # 7
echo "Multiplication: ", a * b # 30
echo "Division: ", a / b # 3.3333333333333335 (float)
echo "Integer division: ", a div b # 3
echo "Modulus: ", a mod b # 1

Important notes:

  • / always returns a float, even with integer operands
  • Use div for integer division
  • Use mod for modulus (remainder) operation
  • Nim also supports compound assignment operators like +=, -=, etc.

Conditions in Nim

Nim uses if, elif (else if), and else for conditional execution. The syntax is similar to Python but uses elif instead of elif.

# Basic conditional structure
let score = 85

if score >= 90:
  echo "Grade: A"
elif score >= 80:
  echo "Grade: B"
elif score >= 70:
  echo "Grade: C"
else:
  echo "Grade: F"

# One-liner condition (ternary-like)
let message = if score > 50: "Pass" else: "Fail"

Key characteristics:

  • Conditions don't require parentheses
  • Blocks are defined by indentation (like Python)
  • You can use the if expression for assignments
  • Nim also supports case statements for pattern matching

Loops in Nim

Nim provides several looping constructs, with for and while being the most common. Nim's loops are powerful and expressive.

For Loops

# Iterate over a range
for i in 0..5:
  echo "Count: ", i

# Iterate over a sequence
let fruits = @["apple", "banana", "cherry"]
for fruit in fruits:
  echo "Fruit: ", fruit

# Count down
for i in countdown(5, 0):
  echo "Countdown: ", i

While Loops

# Basic while loop
var counter = 0
while counter < 3:
  echo "While loop: ", counter
  counter += 1

# Infinite loop with break
while true:
  let input = readLine(stdin)
  if input == "quit":
    break
  echo "You entered: ", input

Important notes:

  • for loops are preferred when the number of iterations is known
  • while loops are used when the condition is dynamic
  • Nim also has block and break for more complex control flow
  • The .. operator creates inclusive ranges
